📊 Formule du score FICT

Le score FICT mesure la priorité de contact d'un élu pour les affaires publiques FICT : un élu fortement ancré dans une zone de production ET actif sur les dossiers agricoles est prioritaire.

score_fict = score_eco × (1 + (score_impl + score_engagement) / 10)

# Score implantation (poids économique de la zone)
score_eco = nb_sites_fict
          + 2 × nb_sites_charcuterie
          + 1 × nb_sites_transformation_viande
          + 0.5 × nb_sites_volaille

# Score implication parlementaire
score_impl = 3 × membre_commission_agriculture
           + 2 × membre_mission_egalim
           + 1 × membre_commission_economique
           + 0.5 × taux_participation

# Score engagement (missions et groupes spécifiques FICT)
score_engagement = Σ memberships (0–11 points max)
    # commission_enquete_gd, mission_nitrites, groupe_etude_iaa...

Géographie : le score utilise le département (et non la circonscription) comme unité d'analyse - précision sub-départementale non disponible via SIRENE.

⚠️ Limites à connaître
  • Niveau département : le score utilise le département (pas la circonscription) - tous les élus d'un département reçoivent le même score d'implantation.
  • Interventions thématiques : les interventions hémicycle (nosdéputés.fr) sont globales, pas filtrées sur les thèmes agricoles.
  • Scrutins Sénat : les votes nominatifs ne sont disponibles que pour les députés (AN). Les sénateurs votent selon des procédures différentes.
  • SIRENE : délai de mise à jour possible (ouvertures/fermetures récentes peuvent manquer).
  • 17e législature en cours : sort: null sur les amendements = normal pour la session en cours (pas encore finalisés).
